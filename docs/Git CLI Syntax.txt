local area
- git init ( initalize the local folder)

staging area 
- git status (see files in staging area)
- git add . or git add <filename>  ( add all files / one file to staging)
- git rm -- cached <fielname>   (remove file from staging)
- git commit -m "message about commit"
OR
- git commit -am " add 'a' stages and commit everything"

- (Branches)
- git checkout -b new-branch (created an switch to new branch)
- git checkout master (switch to master branch)
- git branch ( see branches )

- (Merge)
- git checkout master (switch to master branch)
- git merge <branchname> (merge <branchname> with master )

- (Merge Conflict)
- Remove one
- Add more
- Keep both 
- just modify the file, remove of adjust the file to keep what you need
